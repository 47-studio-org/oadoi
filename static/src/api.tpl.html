API

GET /v1/publication/doi/:doi

Here's the API call to get the oaDOI API results for one doi: http://api.oadoi.org/v1/publication/doi/10.1016/j.tree.2007.03.007?source=YOURTOOL
The source=YOURTOOL is optional, but it helps us usage to report to our funders, so thanks in advance for including either the name of your tool or your email address.

POST /v1/publications

If you are querying for many DOIs, you'll get faster results (and helps us make fewer requests to our data sources) if you call the POST endpoint with a list of DOIs.
curl -X POST -H "Accept: application/json" -H "Content-Type: application/json" -d '{"dois": ["10.6084/M9.FIGSHARE.144", "10.1371/journal.pone.0000308"]}' "http://api.oadoi.org/v1/publications"

Return

These return results objects that look like this: 
    {
        doi: "10.1016/j.tree.2007.03.007",
        doi_resolver: "crossref",
        evidence: "scraping of oa repository (via base-search.net unknown-license url)",
        free_fulltext_url: "https://works.bepress.com/ethan_white/3/download/",
        is_boai_license: false,
        is_free_to_read: true,
        is_subscription_journal: true,
        license: null,
        oa_color: "green",
        url: "http://doi.org/10.1016/j.tree.2007.03.007"
    }
