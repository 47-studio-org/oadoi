API

The REST API allows programmatic access to read oaDOI's data. It's free and open for anyone to use. Just please send ?email=YOUREMAIL in your reqeusts so we can get in touch if something breaks, and so we can report usage to our funders :). The rate limit is 10k requests per day, but get in touch if you need more and we'll hook you up.

Endpoints

GET /v1/publication/doi/:doi

This is handy for testing because you can run it in your browser. Here's an example: 

http://api.oadoi.org/v1/publication/doi/10.1038/ng.3260?email=YOUREMAIL

POST /v1/publications

This is probably the endpoint you want, since you can query up to 25 DOIs at a time this way. That helps you stay inside your rate limit, gets much faster results, and helps us make fewer requests to our data sources. 
            
curl -X POST -H "Accept: application/json" -H "Content-Type: application/json" -d '{"dois": ["10.1038/ng.3260", "10.1371/journal.pone.0000308"]}' "http://api.oadoi.org/v1/publications"

Return format

Here's an example of what you get back. We'll be adding more documentation on on these fields soon. 
{
    doi: "10.1038/ng.3260",
    doi_resolver: "crossref",
    evidence: "scraping of oa repository (via base-search.net oa url)",
    free_fulltext_url: "https://dash.harvard.edu/bitstream/handle/1/25290367/mallet%202015%20polytes%20commentary.preprint.pdf?sequence=1",
    is_boai_license: false,
    is_free_to_read: true,
    is_subscription_journal: true,
    license: "cc-by-nc",
    oa_color: "green",
    url: "http://doi.org/10.1038/ng.3260"
}
