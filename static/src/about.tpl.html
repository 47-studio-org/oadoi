About

oaDOI is an alternative DOI resolver that gets free fulltext where available, instead of just an article landing page.
  • DOI gets you a paywall page: http://doi.org/10.1016/j.tree.2007.03.007
  • oaDOI gets you a PDF: http://oadoi.org/10.1016/j.tree.2007.03.007

Data Sources

We look for open copies of articles using the following data sources:
  • BASE is our main way to find a Green OA full text for a given DOI. It indexes 90mil+ open documents in 4000+ repositories by harvesting OAI-PMH metadata.
  • CrossRef returns license information for resolved DOIs, where publishers have made available (sadly not that frequently). We use this information where it exists.
  • The Directory of Open Access Journals (DOAJ) lists open access journals. We use a local copy of the DOAJ to identify Gold OA DOIs
  • Custom page crawls. This is the main advantage of oaDOI over similar services (see below). Significantly improves coverage, especially for hybrid and Green OA articles.
