Impactstory and ORCID

Auto-generated profiles

ORCID users control all access to their data, but many ORCID users choose to make their data public.

We've used open data from public ORCID profiles to help us auto-generate some Impactstory profiles. These auto-generated profiles help us build robust percentile information, since we can compare any given user to a nice large set of scholars at large.

If you've got an auto-generated ORCID profile, you'll probably want to log in to claim it—it's as simple as logging in to your ORCID account and takes just a few seconds. Once you've logged in, you can get start getting email updates on new online impacts, as well as also modify or delete your profile if you'd like.

If you have any trouble logging in, just drop us a line and we'll be glad to help.
